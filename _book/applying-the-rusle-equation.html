The Revised Universal Soil Loss Equation (RUSLE) for post-fire soil erosion modelling.

Chapter 1 Applying the RUSLE Equation:

In this section, we present two example applications of the RUSLE. These case studies show a applications of the RUSLE in the same region pre and post fire.

1.1 Application 1: RUSLE Post Burn

1.1.1 Step 1: Load RUSLE Factors

K = raster("data/4/K/K_Factor.tif")
R = raster("data/4/R/R_Factor.tif")
LS = raster("data/4/LS/LS_Factor.tif")
C_PreBurn = raster("data/4/C/C_Factor_PreBurn.tif")

1.1.2 Step 2: Set the Extent of RUSLE Factors

extent(K) <- extent(LS)
extent(C) <- extent(LS)
extent(R) <- extent(LS)
extent(C_PreBurn) <- extent(LS)

1.1.3 Step 3: Resample RUSLE Factors

K <-  resample(K, LS)
R <-  resample(R, LS)
C <-  resample(C, LS)
C_PreBurn <-  resample(C_PreBurn, LS)

1.1.4 Step 4: Combine Factors using the RUSLE

RUSLE_PreBurn <- LS*K*R*C
